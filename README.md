# FDCL Homepage

Generated by [Jekyll](https://jekyllrb.com) using [Nomod theme](https://jekyllthemes.io/theme/nomod-blog-jekyll-theme)



### Development

*   Install Jekyll according to [Jekyll Installation](https://jekyllrb.com/docs/installation/)
*   Follow quick tutorial at [Ruby 101](https://jekyllrb.com/docs/ruby-101/)
*   Development
    *   In `_config_yml`, set `baseurl: `
    *   Run `build.sh`
    *   Run `server.sh`
    *   Open brower at [`https://127.0.0.1:4000`](https:/127.0.0.1:4000)

*   Deployment
    *   In `_config_yml`, set `baseurl: /~tylee`
    *   Run `build.sh`
    *   Run `scp_zippy.sh`

    *   Create a new markdown file at `/_members/`
    *   Copy a photo image to `images/student/`

*   To add news articles
